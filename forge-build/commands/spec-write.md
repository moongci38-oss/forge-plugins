# /spec-write (Alias)

이 명령은 `/forge-spec`으로 이름 변경되었습니다. 동일 동작:

```

```

모든 인자, 옵션, 동작은 `/forge-spec`과 100% 동일합니다.
기존 인자 예: `/forge-spec "결제 API 연동" --plan planning/active/payment.md`
