# Assessment: yt

## 테스트 입력

- input_1: "Analyze this YouTube video: https://www.youtube.com/watch?v=example123"

1. 트랜스크립트 추출 단계가 포함되어 있는가?
2. AI 분석(TL;DR, 비판적 분석, 팩트체크) 단계가 계획되어 있는가?
3. GTC 4단계 검증이 언급되어 있는가?
4. ACHCE 축 태깅 또는 시스템 개선 제안이 포함되어 있는가?
5. 저장 경로 또는 Notion 등록이 계획되어 있는가?
